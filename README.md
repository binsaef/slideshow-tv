# 📺 عارض الصور — Android TV

## طريقة البناء عبر GitHub (مجاناً - الأسهل)

### الخطوة 1: رفع المشروع على GitHub
1. سجّل دخول على [github.com](https://github.com)
2. اضغط **New Repository** → اسمه: `slideshow-tv`
3. اضغط **Create repository**
4. ارفع محتوى هذا المجلد كاملاً (drag & drop أو GitHub Desktop)

### الخطوة 2: تفعيل GitHub Actions
1. افتح الـ repository → اضغط تبويب **Actions**
2. سترى workflow اسمه **Build APK**
3. اضغط **Run workflow** → **Run workflow**
4. انتظر 3-5 دقائق

### الخطوة 3: تحميل APK
1. بعد انتهاء البناء (✅)، اضغط على الـ workflow
2. انزل للأسفل → **Artifacts**
3. اضغط **SlideshowTV-release** لتحميل APK
4. ستحصل على ملف `.apk` جاهز للتثبيت

---

## تثبيت APK على Android TV

### تفعيل مصادر غير معروفة:
- إعدادات → أمان → **تثبيت تطبيقات من مصادر غير معروفة** ✅

### طرق التثبيت:

**الطريقة أ - USB:**
1. انسخ APK على فلاشة USB
2. أدخل الفلاشة في TV
3. افتح تطبيق **مدير الملفات** على TV
4. اضغط على ملف APK → **تثبيت**

**الطريقة ب - عبر ADB (إن كان TV على نفس الشبكة):**
```bash
adb connect [IP-of-TV]:5555
adb install SlideshowTV-release.apk
```

## استخدام التطبيق

1. افتح **عارض الصور** من قائمة التطبيقات
2. سيطلب إذن الوصول للملفات → اقبل
3. اضغط **"اختر الصور من الفلاشة"**
4. انتقل لمجلد الصور في الفلاشة
5. حدد الكل → فتح → يبدأ العرض تلقائياً
